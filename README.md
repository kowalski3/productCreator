# product creator tool

### Overview
- Product creator tool allows the user to select an album or list of albums and compile them from unique files into either Bin, CDG, MP4 or XML/MP3 format
- Written in Scala with Processing (java)-based UI

###Dependencies
1. Scala (2.11.7)
2. Processing UI.  Requires core.jar from https://processing.org/download/
3. Processing Library controlP5 http://www.sojamo.de/libraries/controlP5/
